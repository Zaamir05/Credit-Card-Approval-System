Project Overview

About This Project

Automating credit card approval decisions with state-of-the-art Machine Learning — from data collection to IBM Watson cloud deployment.

Project Overview

Banks receive thousands of credit card applications every day. Manual verification is time-consuming, costly, and prone to human bias.

This project automates the credit card approval process using Machine Learning. The system analyses applicant demographic and financial information and predicts whether an application should be Approved or Rejected.

The best-performing model is integrated with a Flask web application and is ready for deployment on IBM Watson Machine Learning.

Project Objectives

  • Automate credit card approval decisions using trained ML models
  • Analyse and preprocess real-world credit application data
  • Train and evaluate Logistic Regression, Decision Tree, Random Forest & XGBoost
  • Select the best model and serialise it with Pickle
  • Build a professional Flask web application for real-time inference
  • Deploy the solution on IBM Watson Machine Learning for enterprise use
  • Ensure a responsive, accessible, and modern user interface

Machine Learning Models

Logistic Regression

Linear baseline model — fast, interpretable, good for linearly separable data.

Decision Tree

Tree-based classification — highly explainable with clear decision paths.

Random Forest

Ensemble of trees — highest accuracy (93.4%), robust to overfitting.

XGBoost

Gradient boosting champion — 91.7% accuracy with feature importance ranking.
